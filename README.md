# DevsBurger - Cliente 🍔

Este é o front-end do sistema de pedidos **DevsBurger**, voltado para os clientes. Aqui, os usuários podem adicionar seus pedidos ao carrinho, inserir informações do pedido (nome do cliente, retirada no local ou delivery e forma de pagamento), e acompanhar o status do pedido em tempo real.
